# Acknowledgements

DualKey Signal Light was inspired by [starlight36/vibecoding-signal-light](https://github.com/starlight36/vibecoding-signal-light), an MIT-licensed project that gives local AI coding agents a physical red/yellow/green status light.

This repository carries that core idea and compact lamp language to different hardware. The firmware, BLE/USB protocol, host bridge, key interactions, and Chain DualKey integration in this repository are a new implementation for the M5Stack Chain DualKey.

Hardware pin definitions, power-control behavior, and the ROM download procedure were derived from M5Stack's public [Chain DualKey documentation](https://docs.m5stack.com/en/chain/Chain_DualKey) and [M5DualKey-UserDemo](https://github.com/m5stack/M5DualKey-UserDemo).

Thank you to the maintainers and contributors of those projects and of the open-source dependencies listed in [THIRD_PARTY_NOTICES.md](THIRD_PARTY_NOTICES.md).
